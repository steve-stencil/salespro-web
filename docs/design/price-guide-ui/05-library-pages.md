## Purpose

The Library pages provide access to the shared entities that can be linked to multiple MSIs:

- **Options** - Product variants (e.g., "Pella Premium", "Andersen 400")
- **UpCharges** - Add-ons and accessories (e.g., "Low-E Glass", "Grilles")
- **Additional Details** - Custom input fields (e.g., "Window Dimensions", "Notes")

This is the "master data" view where users manage these entities independently of MSIs.

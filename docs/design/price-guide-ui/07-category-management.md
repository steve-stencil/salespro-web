## Purpose

Categories organize Measure Sheet Items into a hierarchical structure. Users can:

- Create nested category hierarchies (Windows > Double Hung > Premium)
- Drag-and-drop to reorder and reparent categories
- See item counts per category
- Manage the entire category tree from one interface

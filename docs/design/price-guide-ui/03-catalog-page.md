## Purpose

The Catalog page is the primary entry point for managing Measure Sheet Items (MSIs). It provides:

- Browsable list of all MSIs with search and filtering
- Quick access to view, edit, and manage pricing
- Bulk selection for mass operations
- Import/export functionality

---

## URL

```
/dashboard/price-guide/catalog
```

**Query Parameters:**

- `?search=` - Pre-populate search
- `?category=` - Filter by category ID
- `?office=` - Filter by office ID
- `?status=` - Filter by pricing status (complete, incomplete, none)

### Primary Query

```typescript
// GET /api/price-guide/msis
// Uses cursor-based pagination
{
  search?: string;
  categoryId?: string;
  officeId?: string;
  status?: 'complete' | 'incomplete' | 'none';
  sort?: 'name_asc' | 'name_desc' | 'created_desc' | 'created_asc' | 'category';
  cursor?: string;
  limit: number; // default 50
}

// Response
{
  items: MeasureSheetItemSummary[];
  nextCursor?: string;
  hasMore: boolean;
  total: number;
}
```

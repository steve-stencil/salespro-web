## Purpose

The Item Detail page provides a comprehensive view of a single Measure Sheet Item (MSI), including:

- All linked options, upcharges, and additional detail fields
- Pricing summary across all offices
- Edit capabilities for all aspects of the item
- Audit history of changes

This is the "deep dive" view for understanding and configuring a single product in the price guide.

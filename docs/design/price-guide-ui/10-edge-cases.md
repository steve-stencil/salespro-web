### Performance Requirements

| Operation            | Target  | Approach                               |
| -------------------- | ------- | -------------------------------------- |

| Infinite scroll page | < 500ms | Cursor-based pagination, 50 items      |
| Bulk operation start | < 3s    | Background job, immediate feedback     |

## UpCharge Pricing

UpCharge pricing is more complex because it supports:

1. **Per-price-type mode selection** - Each price type (Materials, Labor, Tax, Other) can independently be Fixed or Percentage
2. **Percentage-based calculations** - Calculate as a percentage of the parent option's price
3. **Selectable percentage base** - Choose which parent option price types to include in the percentage calculation
4. **Option-specific overrides** - Override default pricing for specific options

## Accessibility

### Screen Reader Support

- All cells have aria-labels: "Materials price for Denver, currently $200"
- Grid navigation announced: "Row 2 of 4, Column 2 of 5"
- Changes announced: "Price changed from $200 to $225"
- Warnings read aloud: "Warning: Colorado Springs has no pricing"

### Keyboard Only

- Full grid navigation with arrow keys
- Tab order follows logical left-to-right, top-to-bottom
- Escape always cancels current operation
- Enter confirms and moves to next logical cell
